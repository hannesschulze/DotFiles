# Dotfiles

## Dependencies

- `xfce4`
- `ulauncher`
- `xfce4-appmenu-plugin`
- `alacritty`
- `neovim`
- `plank`
- `inkscape`
- `dconf-cli`
- `gtk2-engines`
- `git`

## Fonts

- Inter
- Jetbrains Mono

## Installation

```sh
python3 Install.py
```

### Completing the installation

Set the desktop background color to **#333**

Set up autostart for the following applications:

- `plank`
- `ulauncher` (can be set up from the application itself)
